# The Test Lab Magazine

A static magazine website built with HTML, inline CSS, and inline SVG. The site includes a front page with five featured articles and individual pages for each story.

## Files included

- `index.html`
- `article1.html`
- `article2.html`
- `article3.html`
- `article4.html`
- `article5.html`
- `PRODUCT.md`
- `ARCHITECTURE.md`
- `CONTRIBUTING.md`

## How to use

Open `index.html` in a browser to view the magazine landing page. Click any article link to navigate to the full story page.

## Publishing

This repository is ready for GitHub Pages as a static site. Ensure `index.html` is present at the project root, then enable Pages from the repository settings.
